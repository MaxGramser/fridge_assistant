# 🖨️ Fridge Assistant – Label Printer

Prints PNG/PDF labels on a USB **DYMO LabelWriter** via CUPS. Companion add-on
to the Fridge Assistant integration, but usable as a generic image/PDF label
printer for anything.

**Tested with a DYMO LabelWriter 550 + 99014 labels (54 × 101 mm).**

See [DOCS.md](DOCS.md) for install, options and the HTTP API.
